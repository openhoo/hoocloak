Run the Helm readiness test:
  helm test {{ .Release.Name }} -n {{ .Release.Namespace }}

The test verifies only in-cluster Service readiness. It does not verify discovery, login, or token issuance.

The configured base_url must be the exact external root URL and must end in /.
Each realm issuer is base_url followed by realms/<realm-name> (for example, /realms/<realm-name> relative to the external root).
Hoocloak is development-only and deliberately restricted to one replica because signing keys and sessions are in memory.
